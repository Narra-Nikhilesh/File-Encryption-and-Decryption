File Encryption & Decryption in C

This project is a  implementation of file encryption and decryption using the C programming language. 
It allows users to securely transform text and files into an encrypted format and then restore them back to their original form.


⚙️ How It Works

The program reads input from a file or user-provided text, applies an encryption algorithm (based on character/string transformation),
and writes the encrypted data into a new file. The decryption process reverses this transformation to retrieve the original content.
